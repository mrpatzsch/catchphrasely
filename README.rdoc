This app is a simple catchphrase app with Ruby/Rails words in the seed.rb file.
1. Fork and Clone this repo
2. Bundle, db:create, db:migrate, db:seed.
3. Run the server and a random word should appear at '/'
4. You can add more words by going to '/words/new'
5. You can see all words by going to '/words/all' --they are in json format
